# Json sorter

Usage
```
seyn -i <json-file>
```

or

```
cat <json-file> | seyn
```

Also you can define output file

```
cat <json-file> | seyn -o <output-file>
```
This flag creates <json-file> file and puts the result in that

---
### Flags
- -i : Define input file
- -o : Define output file
- -r : Reverse sorting
